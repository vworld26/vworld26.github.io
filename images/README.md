# images/

Drop site images here.

- `vernon.jpg` — headshot for the hero. Square works best (the CSS crops to a 180×180 square on desktop, 130×130 on mobile). Anything from ~400×400 to ~1200×1200 is fine; the browser scales it down.
- Client logos (optional): `citi.svg`, `wire.svg`, `exelon.svg`, `provuu.svg`, `clorox.svg`, `visa.svg`. SVG is best, PNG works. When dropped in, swap the corresponding `<span class="logo-mark">…</span>` in `index.html` for an `<img src="images/citi.svg" alt="Citi">`.
